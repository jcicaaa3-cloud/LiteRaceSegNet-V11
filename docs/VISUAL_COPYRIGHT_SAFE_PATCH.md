# Visual Copyright-Safety Patch

This public package removes the previous decorative sparkle-style badge from the Project QA flow diagrams and replaces it with a simple text-only `QA` badge drawn from basic geometric shapes. The change is intentional: the repository should avoid decorative third-party-style logos or icons that could create copyright or trademark confusion.

Affected assets:

- `docs/github_assets/hoshilm_project_qa_flow_diagram.png`
- `docs/github_assets/project_qa_flow.png`

The Project QA module remains a reporting/support extension. It does not generate segmentation masks and does not modify LiteRaceSegNet predictions.
