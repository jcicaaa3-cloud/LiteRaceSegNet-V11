# Dataset and privacy policy

이 저장소는 GitHub 공개 업로드용이므로 다음 파일을 포함하지 않습니다.

- raw road images
- segmentation masks
- private road photos
- pretrained weights
- training checkpoints
- Hugging Face downloaded model weights
- API keys or credentials

데이터셋은 사용자가 사용 권한이 있는 image/mask pair만 `datasets/pothole_binary/processed/` 아래에 배치해야 합니다. 공개 데이터셋을 사용할 경우 해당 데이터셋의 license와 citation을 별도로 확인해야 합니다.
