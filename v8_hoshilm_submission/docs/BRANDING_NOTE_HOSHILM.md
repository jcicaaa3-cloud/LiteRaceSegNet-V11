# Branding Note: HoshiLM-KR

The language-model module in v8 is branded as **HoshiLM-KR**.

Reason:
- Avoid implying association with any commercial chatbot or trademarked product name.
- Present the module as an original, educational decoder-only Transformer language-model lab.
- Keep the claim modest: this is a toy-scale research/learning implementation, not a production assistant or large commercial language model.

Recommended wording:

> HoshiLM-KR is a toy-scale decoder-only Transformer language-model implementation for reproducing the core pipeline of tokenization, causal self-attention, next-token prediction, checkpointing, and sampling.

Avoid wording:

> We built a commercial-scale chatbot.
> This is equivalent to a large production language model.
